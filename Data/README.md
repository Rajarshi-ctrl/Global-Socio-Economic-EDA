# Data Folder

This folder contains the dataset files used for the Global Socio-Economic EDA project.

## Files to Add:

1. EdStatsData.csv - Main dataset with socio-economic indicators
2. EdStatsCountry.csv - Country metadata (Region, Income Group)
3. EdStatsCountry-Series.csv - Country-indicator relationships
4. EdStatsFootNote.csv - Data footnotes
5. EdStatsSeries.csv - Indicator definitions

## Source:
World Bank EdStats Database
https://datacatalog.worldbank.org/
